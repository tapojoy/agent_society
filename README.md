# Multi-Agent Social Simulation Platform

A multi-agent social simulation platform for synthetic social data generation, debate leagues, agent evolution, and emergent behavior research.

## Features

* Formal debate simulation using LLM-powered agents
* Informal agent interactions
* League seasons and knockout tournaments
* Agent evolution through confidence, affinity, and domain specialization
* Persistent agent populations across simulations
* Structured event and state logging
* LangGraph-based workflow orchestration

## Roadmap

* Debate simulation API
* OpenSearch analytics dashboards
* Retrieval-Augmented Generation (RAG)
* Temporal interaction graph construction
* Community evolution prediction
* Infrastructure automation

## Project Status

This repository currently contains the simulation engine. Additional components will be added incrementally as the platform evolves.
